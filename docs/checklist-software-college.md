# 毕业论文常见问题检查清单

## 使用方法

本清单是毕业论文中常见的问题汇总，使用本清单的方法如下：
- 仔细阅读清单中的每条规则，准确理解规则含义
- 清单中的大部分规则应该在论文中进行通篇检查
- 清单中的每条规则都检查过后，再把论文发给老师检查
